LandscapeLab!

an open-source development kit for game-based participatory spatial planning tools
Serious-Gaming Approach
the LandscapeLab! implements simple game mechanics and connects real world objects with user interactions and game data
Human-Computer Interface
the LandscapeLab! projects an interactive map on a table and placed game tokens are translated into game information
Virtual Reality Assessment
any time a participant has the possibility to experience the current situation in the VR landscape generated by the LandscapeLab!
